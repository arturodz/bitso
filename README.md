# Bitso Ruby API


## Installation

Add this line to your application's Gemfile:

```ruby
gem 'bitso'
```

## Create API Key

More info at: [https://bitso.com/api_info](https://bitso.com/api_info)

## Setup

```ruby
agent = Bitso.new(CLIENT, KEY, SECRET)
```
